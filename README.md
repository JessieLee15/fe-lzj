* 作为一个前端 你要学什么
  * 服务器：所有的前端实质，到最后说白了都是资源
  * 操作系统及浏览器：作为前端资源的承载-浏览器，你要去懂他及他的环境
  * 网络：资源、信息的流动
  * 算法：解决问题是每个程序员都要懂的
  * 架构：大部分生产活动，走到后来都要发展工程化
